Vibes...coded.

26 April 2026

So I did it. I vibecoded myself a very minimalist blog using Cursor, hosted on Github pages. It's the first repo I've created on Github, which is bananas, but I'm a data guy who never found the energy to start from zero. There was always a template or an open source repo or something which existed before that I was stealing, forking, or repurposing for my own use...

Which I guess Cursor is also technically doing. But I'm directing it with my vibes, my flava (I can't believe I said that). I'm mostly doing this because I'm curious and I want to learn. I want to figure out a way to self-publish without using the main platforms (although I really like paragraph.com if you're looking for one). And I want to be able to say that I'm able to do this with no one's help other than the AI's...which I guess is technically the help of all humankind being sourced and indexed and regenerated through the AI. But at my bidding...but with its knowledge. Ugh. I digress.

Anyway, at least I can finally say I did it. Or the AI did it with my help. Or vice versa. Whatever.
